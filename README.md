
# 👟 Drip Store

[](https://drip-store-front-mn8k.vercel.app/)
[](https://www.google.com/search?q=https://www.figma.com/design/Ym57X6A8P5O3vF5XyUvVvV/Digital-Store---Drip-Store)
[](https://reactjs.org/)
[](https://tailwindcss.com/)

E-commerce moderno construído com **React** e **TailwindCSS**. O projeto foca em componentização reutilizável, gerenciamento de estado global e interface totalmente responsiva baseada no Figma.

## 🔗 Links do Projeto

  * **🌐 Deploy Oficial:** [Visualizar Loja Online](https://drip-store-front-mn8k.vercel.app/)
  * **🎨 Protótipo Figma:** [Acessar Design do Projeto](https://www.figma.com/design/cfb4F7ZXMFQmvmTn3PKI4z/DRIP-STORE---DIGITAL-COLLEGE?node-id=22-30&p=f&t=qKR6WW85Wtsi4s8W-0)

## 🚀 Funcionalidades

  * **Home:** Carrossel automático e vitrine de produtos em destaque.
  * **Catálogo:** Listagem com filtros funcionais por categoria, marca, gênero e estado.
  * **Produto:** Página de detalhes com seleção de atributos (cor/tamanho) e galeria.
  * **Carrinho:** Gerenciamento de itens com persistência em `localStorage`.
  * **Responsividade:** Layout *mobile-first*.

## 🛠️ Stack Técnica

  * **Core:** React.js (Vite)
  * **Roteamento:** React Router DOM
  * **Estilização:** TailwindCSS
  * **Estado:** Context API
  * **Ícones:** Lucide React / React Icons

## 🔧 Instalação e Uso

```bash
# Clone o repositório
git clone https://github.com/talitalima-tech/drip-store-front.git

# Instale as dependências
npm install

# Inicie o ambiente de desenvolvimento
npm run dev
```
